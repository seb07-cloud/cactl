Create `internal/state/refspec.go`:

- `ConfigureRefspec(repoDir string) error`:
  1. Check if remote "origin" exists via `git config --get remote.origin.url`. If not, return nil (skip silently — no remote to configure).
  2. Check existing fetch refspecs via `git config --get-all remote.origin.fetch`. If already contains `refs/cactl/*`, skip.
  3. Add fetch: `git config --add remote.origin.fetch +refs/cactl/*:refs/cactl/*`
  4. Check existing push refspecs via `git config --get-all remote.origin.push`. If already contains `refs/cactl/*`, skip.
  5. Add push: `git config --add remote.origin.push +refs/cactl/*:refs/cactl/*`
