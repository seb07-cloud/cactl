Create `internal/graph/client.go`:

- `Client` struct with: `baseURL string`, `httpClient *http.Client`, `credential azcore.TokenCredential`, `tenantID string`
- `NewClient(credential azcore.TokenCredential, tenantID string) *Client` — sets baseURL to `https://graph.microsoft.com/v1.0`, httpClient with 30s timeout
- `do(ctx context.Context, method, path string, body io.Reader) (*http.Response, error)` — internal method:
  1. Build full URL from baseURL + path
  2. Create http.Request with context
  3. Acquire token: `credential.GetToken(ctx, policy.TokenRequestOptions{Scopes: []string{"https://graph.microsoft.com/.default"}})`
  4. Set `Authorization: Bearer <token>`, `Content-Type: application/json`
  5. Execute request, return response

Create `internal/graph/policies.go`:

- `Policy` struct: `ID string`, `DisplayName string`, `State string`, `CreatedDateTime string`, `ModifiedDateTime string`, `TemplateID *string`, `RawJSON json.RawMessage` (the full original response body for this policy)
- `ListPolicies(ctx context.Context) ([]Policy, error)`:
  1. GET `/identity/conditionalAccess/policies`
  2. Decode response into `{ Value []json.RawMessage, NextLink string }` struct
  3. For each raw policy in Value: unmarshal to get ID and DisplayName for the Policy struct, but KEEP the full RawJSON for normalization downstream
  4. If `@odata.nextLink` is present, follow it (loop until no more pages)
  5. Return all collected policies
- `GetPolicy(ctx context.Context, policyID string) (*Policy, error)`:
  1. GET `/identity/conditionalAccess/policies/<policyID>`
  2. Return single Policy with RawJSON

<task type="auto">
  <name>Task 2: cactl import command — wire Graph, normalize, and state into full import pipeline</name>
  <files>cmd/import.go, cmd/import_test.go</files>
  <action>
Create `cmd/import.go`:

- `NewImportCmd() *cobra.Command` — registered in root.go's init()
- Flags: `--all` (bool), `--policy` (string, slug or display name), `--force` (bool)
- `runImport(cmd *cobra.Command, args []string) error`:

**Step 1: Validate flags**
- If both `--all` and `--policy` specified, return error
- If `--ci` mode and neither `--all` nor `--policy`, return error (interactive selection not available in CI)

**Step 2: Load config and create dependencies**
- Load config via `config.LoadFromGlobal()`
- Create auth factory via `auth.NewClientFactory(cfg.Auth)`
- Get credential via `factory.Credential(ctx, cfg.Tenant)`
- Create Graph client via `graph.NewClient(cred, cfg.Tenant)`
- Create state backend via `state.NewGitBackend(".")`
- Configure refspec via `state.ConfigureRefspec(".")` (safety net — ensure refspec exists before first import)

**Step 3: Fetch policies from Graph API**
- Call `graphClient.ListPolicies(ctx)`
- If `--policy` flag set: filter to matching policy (match by slug OR display name, case-insensitive). Error if no match.

**Step 4: Read existing manifest**
- Call `state.ReadManifest(backend, cfg.Tenant)`

**Step 5: Process each policy**
For each policy:
1. Derive slug via `normalize.Slugify(p.DisplayName)`
2. Check if slug exists in manifest:
   - If exists AND no `--force`: print skip message with renderer, continue
   - If exists AND `--force`: will overwrite (bump patch version: parse current version, increment patch)
   - If slug exists but maps to different LiveObjectID: error — slug collision detected
3. Normalize RawJSON via `normalize.Normalize(p.RawJSON)`
4. Write to state: `backend.WritePolicy(cfg.Tenant, slug, normalized)` — returns blob SHA
5. Determine version: `1.0.0` for new, patch bump for `--force` re-import
6. Create annotated tag: `backend.CreateVersionTag(cfg.Tenant, slug, version, blobSHA, fmt.Sprintf("cactl import: %s %s", slug, version))`
7. Update manifest entry with all STATE-05 fields:
   - Slug, Tenant, LiveObjectID (from policy.ID), Version, LastDeployed (time.Now().UTC().Format(time.RFC3339)), DeployedBy (from auth mode — "az-cli", "client-secret", etc.), AuthMode, BackendSHA
8. Print success: renderer shows `  + <slug> (v<version>) — <displayName>`

**Step 6: Handle interactive selection (IMPORT-08)**
If neither `--all` nor `--policy`:
1. Compare Graph policies against manifest — find untracked (slug not in manifest)
2. If all tracked: print "All policies are already tracked" and return
3. Print untracked list with `?` sigil: `  ? <displayName> (<id>)`
4. Prompt user: "Import which policies? (all, none, or comma-separated numbers): "
5. Parse selection, import selected policies using same Step 5 logic

**Step 7: Write manifest**
- Call `state.WriteManifest(backend, cfg.Tenant, manifest)`
- Print summary: "Imported N policies for tenant <tenant>"

Register command: Add `rootCmd.AddCommand(NewImportCmd())` in appropriate location (follow init.go pattern).

Keep import_test.go focused on flag validation and command registration. Full integration testing of the import pipeline requires real Graph API access and is deferred to UAT.
