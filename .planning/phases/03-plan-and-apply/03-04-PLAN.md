Create `pkg/types/plan.go` with JSON output types:
- `PlanOutput` struct: SchemaVersion int `json:"schema_version"`, Actions []ActionOutput `json:"actions"`, Summary SummaryOutput `json:"summary"`, Warnings []string `json:"warnings,omitempty"`
- `ActionOutput` struct: Slug string, DisplayName string, Action string (lowercase: "create", "update", "recreate", "untracked"), VersionFrom string (omitempty), VersionTo string (omitempty), BumpLevel string (omitempty), Diffs []DiffOutput (omitempty), Warnings []string (omitempty)
- `DiffOutput` struct: Path string, Type string ("added", "removed", "changed"), OldValue interface{} (omitempty), NewValue interface{} (omitempty)
- `SummaryOutput` struct: Create int, Update int, Recreate int, Untracked int, Noop int

Create `internal/output/diff.go`:
- Add ANSI codes: colorCyan = "\033[36m", colorMagenta = "\033[35m" (extend existing color constants in human.go, or define locally if human.go constants are unexported)
- `sigils` map: ActionCreate -> "+"/green, ActionUpdate -> "~"/yellow, ActionRecreate -> "-/+"/red, ActionUntracked -> "?"/cyan
- `RenderPlan(w io.Writer, actions []reconcile.PolicyAction, validations []validate.ValidationResult, resolver *resolve.Resolver, useColor bool)`:
  1. Print each non-noop action with sigil + slug + version bump info
  2. For ActionUpdate: indent and print each FieldDiff with +/- sigils, old/new values. Use resolver.DisplayName() to annotate GUIDs with names.
  3. For ActionCreate: print "(new policy, initial version 1.0.0)"
  4. For ActionRecreate: print red warning about recreation
  5. For ActionUntracked: print "(untracked: exists in tenant, not in backend)"
  6. SEMV-06: If BumpMajor, print explicit WARNING line in yellow/red
  7. Print validation warnings/errors after actions
  8. Print summary line: "Plan: N to create, N to update, N to recreate, N untracked."
  9. Respect useColor flag for all ANSI output
- `RenderPlanJSON(w io.Writer, actions []reconcile.PolicyAction, validations []validate.ValidationResult, resolver *resolve.Resolver)`:
  1. Convert actions to ActionOutput slice
  2. Build SummaryOutput from action counts
  3. Set SchemaVersion to 1
  4. Marshal PlanOutput with json.MarshalIndent, write to w

Create `cmd/plan.go`:

Register `planCmd` on rootCmd in init(). Command: `cactl plan`. No positional args.

`runPlan(cmd *cobra.Command, args []string) error` flow:
1. Load config via `config.LoadFromGlobal()`
2. Validate tenant is set (required for plan). If not, return ExitError code 2 with message "tenant is required: use --tenant or set CACTL_TENANT"
3. Create auth factory and credential via `auth.NewClientFactory()` and `factory.Credential(ctx, cfg.Tenant)`
4. Create graph client: `graph.NewClient(cred, cfg.Tenant)`
5. Create git backend: `state.NewGitBackend(".")`
6. Load backend policies: iterate `backend.ListPolicies(cfg.Tenant)`, for each slug call `backend.ReadPolicy()`, unmarshal JSON to `map[string]interface{}`
7. Load live policies: `graphClient.ListPolicies(ctx)`, for each normalize via `normalize.Normalize(policy.RawJSON)`, unmarshal to map. Build `map[string]LivePolicy` keyed by policy ID.
8. Load manifest: `state.ReadManifest(backend, cfg.Tenant)`
9. Reconcile: `reconcile.Reconcile(backendPolicies, livePolicies, manifest)`
10. Compute semver bumps: load semver config from viper (semver.major_fields, semver.minor_fields with defaults), for each ActionUpdate call `semver.DetermineBump(action.Diff, majorFields, minorFields)`, compute version string via `semver.BumpVersion(currentVersion, bumpLevel)`
11. Run validations: build ValidationConfig from viper (validation.break_glass_accounts), call `validate.ValidatePlan(actions, validationCfg)`
12. Resolve display names: `resolve.CollectRefs(actions)`, `resolver.ResolveAll(ctx, refs)`
13. Render: check `cfg.Output` -- if "json" call `output.RenderPlanJSON(os.Stdout, ...)`, else call `output.RenderPlan(os.Stdout, ...)`
14. Exit code: if any actionable changes (create/update/recreate), return `&types.ExitError{Code: types.ExitChanges, Message: "changes detected"}`. If validation errors (SeverityError), return ExitError code 3. Otherwise return nil (exit 0).

Create `cmd/plan_test.go`:
- Test planCmd is registered on rootCmd (cmd.Name() == "plan")
- Test plan requires --tenant flag (returns error without it)
- Test plan command has no unexpected flags

Run: `go test ./cmd/ -run TestPlan -v` and `go build ./...`
